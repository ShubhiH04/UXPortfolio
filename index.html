Spotify Music Analytics

Problem

Spotify Product and Marketing Teams and Record Labels need an interactive way to explore Spotify track data, understand trends in popularity, streams, genres, and musical keys, and make data-driven insights. Existing tools lacked interactive visualizations with clear UX for filtering and comparison.

Role & Process

Led end-to-end dashboard design: structured KPIs, built interactive Streamlit components, selected and applied intuitive color palettes, designed side-by-side bar charts for clarity, and ensured responsive layouts for usability. Iterated based on feedback from 5+ usability tests and accessibility considerations.

Solution

An interactive Streamlit dashboard visualizing Spotify track popularity, streams, genres, and musical keys over time. Features include KPI summaries, side-by-side comparisons of major/minor keys, and distinct color palettes for categories to enhance clarity and user interpretation.

Impact

Enabled users to quickly identify trends in Spotify data and draw insights on track popularity and genre trends. Strengthened skills in UX for data visualization, interactive dashboard design, and integrating design with functional Python code.

Research

Research • Machine Learning • UX/Product Development

When AI Lies With Confidence

Problem

LLMs generate convincing but inaccurate health information, particularly in women's reproductive health, where confident misinformation can be harmful.

Role & Process

Sole developer on a 4-person team: built infrastructure in Python and LangChain, analyzed 4,000+ LLM responses, scraped NIH databases, and trained a RandomForestClassifier to identify confidence vs. accuracy.

Solution

Machine learning pipeline scoring LLM responses on confidence and accuracy, visualized in a trust matrix via a Chrome extension. Built with Flask, HTML, and JS; integrated NIH fact-checking for real-time user alerts.

Impact

Model achieved 90% accuracy in predicting misinformation. Findings presented at iSchool Research Showcase, earning a Visionary Lens Award. Tool empowers users to critically evaluate AI health advice, mitigating misinformation risks.
